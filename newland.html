Step 1 — Open Scanner Settings

For Newer Devices

Step 1.1: Search for Quick Settings app

Step 1.2: Open the app → Select Scan

For Older Devices

Step 1.1: Open Settings → Extended Settings

Step 1.2: Go to Settings → Extended Settings → Scan Settings

Step 2 — Configure Scanner

Step 2.1: Enable Scan

Step 2.2: Set Output Mode → Output via API

Step 2.3: Configure Broadcast Output Settings

Broadcast Action: com.edgewise.handheld.ACTION
Result Extra: com.symbol.datawedge.data_string

Step 3 — Enable EAN-13 Check Digit (Important)

If you are scanning EAN-13 barcodes, the check digit must be enabled.

Step 3.1: Open the Quick Settings app

Step 3.2: Select Scan

Step 3.3: Tap Symbologies

Step 3.4: Select EAN-13

Step 3.5: Enable Transmit Check Digit

✅ This ensures the complete barcode value is transmitted to the app.
